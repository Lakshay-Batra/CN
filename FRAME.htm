Data Link Layer

  • Service provided by physical layer bit delivery

  • Service provided to network layer framing

  • error control

  • flow control

  • medium access (with shared medium)

Framing

  • Mapping stream of physical layer bits into frames

  • Mapping frames into bit stream

  • Frame boundaries can be determined using: Character Counts

  • Flags

  • CRC Checks

Framing methods:

Four methods:

Ø      Character count

Ø      Flag bytes with bytes stuffing

Ø      Starting & ending flags with bit stuffing

Ø      Physical layer coding violations
