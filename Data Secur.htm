DATA SECURITY

AEP Networks delivers a policy networking suite from the edge to the core. Policy networking ties network and application access to identity and policy, a necessity arising from the dissolution of the traditional network perimeter.


Security needs:

i) Network access control based on user identity

ii) Policy based resource control based on user identity

iii) Cut down on the threat of self-propagating attacks by users (remote, mobile, wireless, satellite, LAN/WAN)

iv) Control unmanaged devices

v) Increased regulatory compliance through the use of identity

 

 

 

1. Both remote and LAN-based users are identified and machines are scanned.

2. Data can be encrypted in transit
“on the fly” for privacy.

3. Controlled application access:
Users only see/access as they are authorized. Individual packets are continuously identified and tracked.

4. Policies enforced at multiple layers.

5. Reporting and logging for compliance requirements

Build and Maintain a Secure Network

Requirement 1: Install and maintain a firewall configuration to protect data.

Requirement 2: Do not use vendor-supplied defaults for system passwords and other

security parameters.

Protect Cardholder Data

Requirement 3: Protect stored data.

Requirement 4: Encrypt transmission of cardholder data and sensitive information across

public networks.

Maintain a Vulnerability Management Program

Requirement 5: Use and regularly update anti-virus software.

Requirement 6: Develop and maintain secure systems and applications.

Implement Strong Access Control Measures

Requirement 7: Restrict access to data by business need-to-know.

Requirement 8: Assign a unique ID to each person with computer access.

Requirement 9: Restrict physical access to cardholder data.

Regularly Monitor and Test Networks

Requirement 10: Track and monitor all access to network resources and cardholder data.

Requirement 11: Regularly test security systems and processes.

Maintain an Information Security Policy

Requirement 12: Maintain a policy that addresses information security.
